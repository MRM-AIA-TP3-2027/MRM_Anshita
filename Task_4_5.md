# Task 4 & 5: Branching and Merging

## Task 4:

In this task, I created a new branch called **Temp** and added a file Task_3.md describing Task 3.

### Commands used:

* git checkout -b Temp
* git add .
* git commit -m "Added Task_3.md in Temp branch"
* git push origin Temp

## Task 5:

In this task, I merged the Temp branch with the main branch.

### Commands used:

* git checkout main
* git merge Temp
* git add .
* git commit -m "Merged Temp branch into main"
* git push
